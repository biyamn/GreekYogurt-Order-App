# 그릭요거트 주문 앱
## 📜 프로젝트 소개
토핑을 선택하여 장바구니에 추가하고 주문할 수 있는 그릭요거트트 주문앱입니다.

<br>

## 💡 주요 페이지 및 기능
각 카테고리에서 각기 다른 종류의 그릭요거트 토핑을 선택 후 수량을 지정하여 장바구니에 추가할 수 있습니다. 
장바구니 버튼을 누르면 모달이 뜨며 담은 재료의 상세 정보와 총액을 확일할 수 있습니다.

<br>
